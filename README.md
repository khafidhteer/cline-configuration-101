## Quick Start

```bash
# 1. Install Cline CLI
npm install -g cline

# 2. Configure OpenRouter
cline auth --provider openrouter -k YOUR_API_KEY -m deepseek/deepseek-chat

# 3. Run your first task
cline "build a landing page"
```

Or [install Cline in VS Code](LEVEL-1-QUICK-START/guides/01-install-vscode.md) for a graphical interface.
