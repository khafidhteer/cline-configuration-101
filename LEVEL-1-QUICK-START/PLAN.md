# 🟢 Level 1: Quick Start — Get Cline Working Today

[⬅ Back to Home](../GETTING-STARTED.md) | [📖 Full Roadmap](../ROADMAP.md)

---

**Time Investment:** ~30 minutes  
**Goal:** Install Cline, configure OpenRouter, run your first task  
**Cost Strategy:** Use cheap/free models (don't spend money yet)

---

## Learning Objectives

By the end of this level, you will be able to:

- [ ] Install Cline in VS Code
- [ ] Install the Cline CLI (optional)
- [ ] Configure OpenRouter as your AI provider
- [ ] Run your first task with Cline
- [ ] Understand when to use Plan vs Act mode
- [ ] Know when to use VS Code vs CLI

---

## Prerequisites

- A computer with internet access
- An email account (to sign up for OpenRouter)
- Basic familiarity with your computer's terminal/command prompt

---

## Action Items

### Step 1: Install VS Code (5 minutes)
- [ ] Download VS Code from [code.visualstudio.com](https://code.visualstudio.com)
- [ ] Install the Cline extension from the VS Code marketplace
- [ ] Follow the guided walkthrough

📖 [Install VS Code Guide →](guides/01-install-vscode.md)

### Step 2: Install CLI (5 minutes — optional)
- [ ] Open your terminal
- [ ] Run `npm install -g cline`
- [ ] Verify installation with `cline --version`

📖 [Install CLI Guide →](guides/02-install-cli.md)

> **Note:** You can skip this step if you only want to use VS Code. Come back when you're ready for automation.

### Step 3: Setup OpenRouter (10 minutes)
- [ ] Create an OpenRouter account at [openrouter.ai](https://openrouter.ai)
- [ ] Generate an API key
- [ ] Configure Cline with your API key
- [ ] Test the connection

📖 [Setup OpenRouter Guide →](guides/03-setup-openrouter.md)

### Step 4: Run Your First Task (5 minutes)
- [ ] Write your first prompt
- [ ] Watch Cline work
- [ ] Review the output
- [ ] Understand the cost

📖 [Your First Task Guide →](guides/04-your-first-task.md)

### Step 5: Understand Plan vs Act (5 minutes)
- [ ] Learn when to use each mode
- [ ] Try a simple plan session
- [ ] Switch to Act mode

📖 [Plan vs Act Guide →](guides/05-plan-vs-act.md)

### Step 6: VS Code vs CLI (2 minutes)
- [ ] Understand the difference
- [ ] Know when to switch
- [ ] Get a feel for both

📖 [VS Code vs CLI Guide →](guides/06-vscode-vs-cli.md)

---

## Verification Checklist

After completing this level, confirm you can:

- [ ] ✅ Open Cline in VS Code and see it working
- [ ] ✅ Send a prompt and get a response
- [ ] ✅ Switch between Plan and Act modes
- [ ] ✅ See the estimated cost in the task header
- [ ] ✅ Open the terminal and run `cline --version`

## Key Takeaways

- ✅ Cline is ready to use in under 30 minutes
- ✅ OpenRouter gives you access to many models with one API key
- ✅ Start with cheap models (DeepSeek, Haiku) — not expensive ones
- ✅ Use Plan mode for complex tasks, Act mode for execution
- ✅ VS Code is for daily development; CLI is for automation
