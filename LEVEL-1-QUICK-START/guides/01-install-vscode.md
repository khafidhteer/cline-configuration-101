**Time:** 5 minutes  
**Goal:** Install VS Code and the Cline extension

---

## What You're Doing

VS Code is a code editor (like Microsoft Word, but for code). The Cline extension adds an AI coding assistant inside it.

## Step 1: Download VS Code

1. Go to [code.visualstudio.com](https://code.visualstudio.com)
2. Click the **Download** button (it should auto-detect Windows)
3. Run the installer — just click "Next" through everything
4. Open VS Code when installation finishes

## Step 2: Install the Cline Extension

1. Open VS Code
2. Click the **Extensions icon** on the left sidebar (it looks like 4 squares: ▦)
   - Or press `Ctrl + Shift + X` on your keyboard
3. In the search bar at the top, type: `Cline`
4. Find the extension called **"Cline"** by Suprati (it should be the first result)
5. Click **Install**
6. Wait a few seconds for it to install

## Step 3: Verify Installation

1. After installation, you should see a new icon on the left sidebar — it looks like a robot face 🤖 or a "C" icon
2. Click that icon to open the Cline panel
3. If you see a welcome screen, you're good!
